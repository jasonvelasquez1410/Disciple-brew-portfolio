# Disciple Brew - 'Faith' AI Digital Assistant

## 🚀 Project Overview

A high-fidelity interactive AI demonstration developed for **Disciple Brew**, a specialty coffee shop. This platform features 'Faith,' a custom-engineered AI persona designed to handle customer engagement with a warm, empathetic tone. The system recreates a pixel-perfect Facebook Messenger interface, providing a familiar and seamless user experience.

[Live Demo Link](https://messenger-ai-fullstack-disciplebrew.vercel.app/)

## 🏗️ Architectural Highlights

**Engine**: Built with **FastAPI** (Python) for high-performance backend logic and **React** (Vite) for a fluid, responsive frontend.

**AI Persona Engineering**: Utilizes **Gemini 1.5 Flash** with advanced system prompting to maintain a consistent 'Faith' persona, capable of Taglish conversation and deep brand knowledge.

**Empathy-First Logic**: Specialized handling for prayer requests and community support, integrating biblical wisdom into customer interactions to reflect brand values.

**Messenger-Inspired UI**: A custom-built, mobile-responsive chat interface using **Tailwind CSS**, featuring real-time message states and automated scrolling.
